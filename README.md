# Supreme-Online-Market

**프로젝트 설명**  

이 프로젝트는 수요가 많은 상품을 주로 제공하는 마켓을 위한 백엔드 시스템입니다. 특히 주문이 많이 몰릴 것으로 예상 되는 시점이고, 추후 백엔드 팀에 여러 개발자들이 합류할 것 이라는 가상의 시나리오를 바탕으로 개발 하였습니다.

## 기능
- **사용자 관리**: 회원가입, 로그인, 프로필 관리
